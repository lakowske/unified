Update your profile, change password, and manage preferences.

🌟 Multi-Service Integration

This unified authentication system supports multiple services:

  • Web Access: Apache with database authentication (current)
  • Email Services: Dovecot/Postfix with role-based access
  • File Sharing: WebDAV with per-user quotas
  • Future Services: Samba, LDAP, and custom applications

👤 User Roles & Permissions

The system supports different user roles with varying access levels:

  • Admin: Full access to all services and administrative functions
  • User: Standard access to web, email, and file services
  • Customer: Limited access (web and file storage, no email)

Your specific permissions are determined by your role assignment in the database.

🛡️ Security Features

  • ✅ Database-backed authentication
  • ✅ Role-based access control
  • ✅ Support for secure password hashing (BCRYPT, SSHA)
  • ✅ Account status management (active/inactive)
  • ✅ Email verification workflow
  • ✅ Audit logging and tracking
